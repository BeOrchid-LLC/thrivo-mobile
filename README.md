# thrivo-mobile

React Native + Expo mobile app for Thrivo.

**Stack:** React Native · Expo · zustand · TanStack Query · TypeScript

**Metro bundler:** `http://localhost:8081`

## Getting started

```bash
git init
npm install
npx expo start
npx expo run:android
npx expo run:ios
```

See `../CLAUDE.md` for conventions and stack details.
